# User Prompt: Market Scan

Analyze the {{ market_type }} market for the following {{ asset_type }}:
{{ assets }}

## Analysis Mode
{{ analysis_mode }}

## Output Requirements

Based on the analysis mode, provide:

### For Crypto Markets:
1. Current trend and momentum (price action, volume)
2. Key support and resistance levels
3. Funding rate status and directional bias
4. Liquidation cluster analysis
5. Whale activity and exchange flows
6. BTC dominance and correlation analysis
7. Trading opportunities with specific entries

### For Stock Selection:
1. Factor scoring (technical/fundamental/macro)
2. Multi-factor model prediction
3. Industry classification and peer comparison
4. Risk-adjusted return metrics (Sharpe, Beta)
5. Portfolio optimization recommendations
6. Position sizing based on confidence

### For Timing Strategy:
1. MA crossover signals (Granville rules)
2. MACD DIF/DEA analysis
3. Trend strength (Hurst index assessment)
4. Momentum confirmation indicators
5. Entry/exit trigger conditions

## Always Provide:
1. Overall market sentiment (Bullish/Bearish/Neutral)
2. Strategy recommendation (Direction/Entry/Target/Stop)
3. Confidence level breakdown
4. Risk factors and invalidation conditions
5. Position sizing suggestion
